# 🛵 Yemeksepeti Bölgesel Restoran Analizi

> Yemeksepeti arama sorgularına bağlı olarak bölgesel restoran envanterini (Pizza, Döner vb.) listeleyen botlar.

## 🧩 Teknik Detaylar
- Sayfalama (Pagination) dinamiğini simüle ederek bütün satıcıların taranmasını sağlar.
- İşletmeye dair "Açık/Kapalı" durumu, "Gönderim Süresi" gibi anlık değişebilen dinamik verileri başarıyla indeksler.
