# 🏷️ Trendyol Bölgesel Restoran Analizi

> İl/ilçe veya koordinat bazlı Trendyol Yemek satıcı analizleri yapan kazıyıcı botlar.

## 📈 Çıktı Formatları
Modül çalıştırıldığında işletme verileri karmaşık web nesnelerinden çıkarılır ve aşağıdaki formatlarda sunulur:
- **Excel (`.xlsx`)**: Analistlerin hızlı okuyabilmesi ve Excel makrolarında işlem görebilmesi için.
- **Metin (`.txt`)**: Ham loglama ve debug amaçlı yapı.
