# 🍔 Yemeksepeti Yorum Entegrasyonu

> Yemeksepeti üzerindeki satıcı restoran profillerine girerek müşteri puanlamaları, yorum metinleri ve tarih verilerini toplayan modül.

## ⚙️ İşlevsellik
- Yemeksepeti'nin güncel arayüzüne uyumlu dinamik sayfa gezintisi.
- Müşterilerin lezzet, hız ve servis puanlarının kantitatif analizi için ayrıştırılması.
- Çeşitli kategorilerdeki (`döner`, `pizza`) yorumların kategorize edilerek CSV dosyalarına aktarılması.

## 🚀 Çalıştırma
Script'ler standart ortamda terminal üzerinden veya IDE aracılıyla çalıştırılabilir. Selenium WebDriver'ın (Örn: ChromeDriver) kurulu ve sistem yoluna (PATH) ekli olduğundan emin olun.
